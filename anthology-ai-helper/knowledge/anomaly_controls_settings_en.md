# English quick guide: Anomaly / Anthology controls and settings

## Third-person camera

Third-person view is switched with camera keys:

- Left Arrow — cam_1
- Down Arrow — cam_2
- Right Arrow — cam_3

Camera zoom is usually T, zoom out is ].

If it does not work, open Settings -> Controls -> Camera and check cam_1, cam_2 and cam_3 bindings.

## Basic controls

- W/A/S/D — movement
- Space — jump
- Left Ctrl — crouch
- Left Shift — sprint
- F — use / interact
- I — inventory
- P — active tasks
- Tab — status / PDA table
- Esc — menu
- F5 — quicksave
- F9 — quickload
- F1-F4 — quick item slots

## Modded hotkeys and MCM

MCM is also part of the game settings for modded features.

Common path:

MCM -> MCM MENU -> All assigned keys

Weapon jam/unjam is usually configured in:

MCM -> WPO -> WPO weapon -> unjam key

If F conflicts with use/search actions, set a separate press mode, for example double tap.

## Flashlight, NVG and detector

- L — flashlight
- N — night vision / NVG
- O — detector

If a key does nothing, check that the required item is equipped and check keybind conflicts.

## Performance and FPS

For FPS drops, first try disabling the heavy graphics mods:

- [GFX] Enhanced Shaders & Color Grading
- [GFX] Beefs NVGs Shaders
- [GFX] ScreenSpaceShaders Update 23.5

Also reduce grass, shadows, view distance and shaders.

For microstutters, use FreeSync/G-Sync if available and cap FPS 1-2 frames below monitor refresh rate.

## MO2 profiles

Switching from Standard to Hard is allowed at any time.

Switching back from Hard to Standard is not safe and can break saves. Do it only with a new game.
