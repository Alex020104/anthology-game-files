# Полные источники для Юры

Кидай сюда полные тексты гайдов в формате `.md` или `.txt`.

Правила:

- один файл = один большой источник или один сюжет/раздел;
- лучше начинать файл с понятного заголовка, например `# Зов Припяти — Затон — Глухарь`;
- текст можно вставлять полностью, без ручного сокращения;
- Юра сначала ищет ответ здесь, и только потом в старых кратких индексах.
