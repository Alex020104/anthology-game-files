# Установка, скачивание и обновление Anthology

## Как скачать игру с последними обновлениями

Если игрок спрашивает, откуда скачать игру или как получить актуальную версию, отвечай так:

1. Скачать основную базу модпака по ссылкам из Discord-сервера Anthology.
2. Скачать актуальный `AnomalyLauncher.exe` из закреплённого сообщения.
3. Скопировать лаунчер в папку:
   `ANTHOLOGY\Anomaly-1.5.3-Anthology 2.1`
   с заменой старого файла.
4. Запустить лаунчер и нажать обновление.
5. Если нужны оружейные патчи или дополнительные инструкции, смотреть соответствующие закрепы Discord.

Не выдумывай прямые ссылки на скачивание. Если точной ссылки нет в базе знаний, говори игроку открыть Discord Anthology и смотреть закрепы/канал со ссылками.

## Переход с Anthology 2.0 на 2.1

Версия 2.1 стала standalone-установкой:

- больше не нужно отдельно скачивать чистую платформу Anomaly;
- сборка ставится единым установщиком;
- версии разделены на «Оригинал» и «Модпак».

## Что делать, если 7-Zip выдаёт ошибку 2

Причина обычно в том, что архив с Яндекс Диска скачался не полностью или повредился в процессе загрузки.

Даже если браузер показывает 100%, файл может ещё иметь временное расширение `crdownload` или не быть полностью сброшен на диск.

Решение:

1. Дождаться полного завершения загрузки, пока файл не примет нормальный вид.
2. Собрать все 3 файла архива в одной папке.
3. Убедиться, что в названиях нет лишних символов.
4. Только после этого запускать распаковку/установку.

## Что делать, если не обновляется лаунчер

Возможные причины:

- нет связи с GitHub;
- GitHub временно недоступен;
- провайдер/регион/VPN мешает загрузке;
- антивирус или защитник Windows блокирует файл;
- старый `Chernobyl Relay Chat.exe` или другой файл ещё запущен и не даёт заменить себя.

Решение:

1. Закрыть игру, лаунчер и Chernobyl Relay Chat.
2. Попробовать обновление снова.
3. Если ошибка связана с GitHub — включить/выключить VPN, сменить страну или сервер VPN.
4. Если ошибка `Permission denied` на `Chernobyl Relay Chat.exe`, значит чат ещё запущен в фоне. Его нужно закрыть в диспетчере задач.

## Permission denied при обновлении Chat Relay

Если лаунчер пишет:

`Permission denied: ... Chernobyl Relay Chat.exe`

значит Windows не даёт заменить файл, потому что Chernobyl Relay Chat всё ещё запущен.

Решение:

1. Закрыть окно Chernobyl Relay Chat.
2. Проверить диспетчер задач и завершить процесс, если он остался.
3. Повторить обновление через лаунчер.

## Игрокам нужен актуальный лаунчер

Для обновлений игровых файлов через `game_packages` игрокам нужен новый лаунчер с поддержкой игровых пакетов. Если у игрока очень старая версия лаунчера, он может не увидеть новые игровые обновления.
