# Профили MO2, хоткеи MCM и важные настройки

## Важное предупреждение по профилям MO2

С профиля **Стандарт** на **Хард** можно переключаться в любой момент.

С профиля **Хард** обратно на **Стандарт** переключаться нельзя.

Обратный переход с Харда на Стандарт может вызвать краши игры и поломку сохранений. Делать такой переход можно только с новой игрой.

## Настройки горячих клавиш в MCM

Основные горячие клавиши настраиваются через MCM.

- Колесо быстрых действий: `MCM -> MCM MENU -> Все назначенные клавиши -> OAW: Колесо быстрых действий -> Зажать`.
  По умолчанию: `MOUSE5`. Используется для быстрого доступа к медицине и еде.

- Клавиша прямой зарядки: `MCM -> MCM MENU -> Все назначенные клавиши -> Клавиша прямой зарядки`.
  По умолчанию: `MOUSE4`. Заряжает магазины в один клик из разгрузки. Работает только при включённом аддоне `[WPN][MAG][R.A.K Weapon Pack Adaptation Global A.N.T.H.O.L.O.G.Y Mags Redux]`.

- Осмотр оружия и устранение клина: `MCM -> MCM MENU -> Все назначенные клавиши -> WPO: Осмотр / устранить клин`.
  По умолчанию: `F`. Важно: ванильное действие обыска тоже может быть на `F`, поэтому для разделения действий лучше изменить настройку.

- Переключение мини-карты: `MCM -> Переключатель mini-карты -> Переключатель mini-карты -> Клавиша переключения mini-карты`.
  По умолчанию: `Z`. Для постоянного отображения карты выберите режим нажатия **Удержание**.

- Смена сеток и режимов ПНВ в прицелах: `MCM -> Переключение меток -> Переключатель меток в коллиматорных прицелах -> Переключение метки коллиматорного прицела`.
  По умолчанию: `BACKSPACE`. Меняет сетки коллиматоров и включает/выключает ПНВ в тепловизионные режимы на продвинутой оптике.

- Проверка патронов и анимация оружия: `MCM -> Проверка патронов -> Проверка патронов`.
  Нужно назначить удобную кнопку для клавиши проверки патронов. По умолчанию: `MINUS` или клавиша `-`.
  Пункт **Исправление "занятых рук"** обязательно нужно отключить галочкой, потому что этот мод включает красивую анимацию проверки магазина, а также включает анимацию осмотра оружия по клавише `F`.

- Клавиша броска светящихся палочек ХИС: `MCM -> Химический источник света -> GLOWSTICKS -> Клавиша броска`.
  По умолчанию: `NUMPAD5`. Нужно назначить кнопку для броска светящейся палочки и выбрать удобный режим нажатия, например **Удержание**.

- Снятие и установка фильтра противогаза: `MCM -> MCM MENU -> Все назначенные клавиши -> Клавиша снятия / установки фильтра`.

- Клавиша снятия клина: `MCM -> WPO -> WPO оружие -> Клавиша снятия клина`.
  По умолчанию: `F`. Лучше выбрать режим нажатия, например **Двойное нажатие**, чтобы случайно не прерывать стрельбу.

- Включение лазерного целеуказателя ЛЦУ: `MCM -> Лазеры на основе BaS -> Лазеры на основе BaS -> Клавиша включения лазера`.
  По умолчанию: `L`. Рекомендуется выбрать режим нажатия **Удержание**.

- Сброс зависшего задания: `MCM -> Job to be done - Redone -> Job to be done - Redone -> Настройки отмены заданий`.
  По умолчанию: клавиша `END` вместе с `Shift`. Чтобы отменить сломанный квест, сделайте его активным в КПК, наведите на него курсор и нажмите `Shift + END`.

## Работа с метками на карте — аддон PAW

Чтобы получить возможность полноценно ставить и удалять маркеры на карте через контекстное меню:

`MCM -> PAW -> Контекстное меню`

В этом разделе нужно изменить значения на **Всегда** для четырёх пунктов:

1. `Настройки точек маршрута -> Всегда`
2. `Настройки маркера -> Всегда`
3. `Добавить маркер -> Всегда`
4. `Удалить маркер -> Всегда`

## Настройка медицины и HUD здоровья — BHS

- Стиль HUD: `MCM -> Body Health System -> HUD -> Тип HUD`.

- Позиция HUD по осям X / Y: `MCM -> Body Health System -> HUD -> Позиция HUD по оси X / Позиция HUD по оси Y`.
  Важно: после ввода числовых значений обязательно нажмите `Enter` прямо в строке, чтобы игра их зафиксировала.

## Контроль анимаций персонажа — погружение

Все параметры находятся по пути:

`MCM -> Animat - Анимации`

Дальше выбирайте нужную вкладку:

- Рюкзак: `Рюкзак -> Включить анимацию`. По умолчанию включено.
- Разделка мутантов: `Разделка мутантов -> Включить анимацию`. По умолчанию включено.
- Анимация подбора: `Анимация подбора -> Включить анимацию`. По умолчанию включено.
- Обыск тел: `Обыск тел -> Включить анимацию`. По умолчанию включено.
- Отключение лишней анимации шлема: `Головные уборы -> Режим строгих шлемов`.
  Снимите галочку, чтобы персонаж автоматически не проигрывал долгую анимацию снятия или надевания шлема каждый раз, когда он ест, пьёт или принимает таблетки.

## Дополнительные опции оружия и интерфейса

- Опции клина оружия: `MCM -> WPO -> WPO оружие`.
  Включите галочками три важные механики:
  - `Разрешить "супер-клин"`
  - `"Выброс" магазина из оружия при "супер-клине"`
  - `Старые боеприпасы влияют на деградацию`

- Стиль HUD: `MCM -> Выбор худа -> Стиль худа`.

- Шрифт текста состояния деталей: `MCM -> MCM MENU -> Основные настройки -> Шрифт текста о состоянии деталей`.
